# Worm rpg
## About

### Description
Worm-rpg is a fan-led online text-rpg, taking place in [worm](https://parahumans.wordpress.com/) web series multiverse.
This is a [MUD](https://en.wikipedia.org/wiki/Multi-user_dungeon), developed with [evennia](https://www.evennia.com/) python framework.

### Status
Currently, project is under-development. Not even an alpha stage.
